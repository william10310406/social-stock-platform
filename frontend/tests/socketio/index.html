🔌 Socket.IO 測試套件

這個測試套件用於驗證 Stock Insight Platform 的 WebSocket/Socket.IO 功能。

基本的 Socket.IO 連接測試，驗證客戶端能否成功連接到後端， 發送 ping 並接收 pong 響應。適合快速驗證系統狀態。
開始測試 正常
🔧 詳細功能測試
完整的 Socket.IO 功能測試，包括連接管理、事件處理、 聊天室功能、錯誤處理等。提供詳細的調試信息。
開始測試 正常
💬 實際聊天功能
測試實際的聊天功能，包括用戶連接、消息發送接收、 在線狀態管理等實際應用場景。
測試聊天 正常
📊 完整功能測試
執行所有 Socket.IO 相關的功能測試，包括單元測試、 集成測試和端到端測試。
完整測試 正常

🔧 技術信息

  • 後端： Flask-SocketIO 5.3.6 + python-socketio 5.8.0
  • 前端： Socket.IO Client 4.0.1
  • 傳輸： polling (HTTP 長輪詢)
  • 伺服器： Gunicorn + Eventlet Worker
  • 協議： Engine.IO 4, Socket.IO 5

🚨 故障排除

如果遇到連接問題：

  1. 確認後端容器正在運行：docker-compose ps
  2. 檢查後端日誌：docker-compose logs backend
  3. 驗證端口映射：後端 5001 → 容器 5000
  4. 測試基本 API：http://localhost:5001/api/health
